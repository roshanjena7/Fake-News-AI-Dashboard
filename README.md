# 🧠 Fake News AI Dashboard

AI-powered Fake News Detection Web Application with Modern Analytics Dashboard.

## 🚀 Features

✅ Machine Learning Fake News Detection
✅ AI Explanation System
✅ Keyword Highlighting
✅ Live News API Integration
✅ Admin Panel with Authentication
✅ Interactive Dashboard & Charts
✅ Modern UI with Animations
✅ Responsive Design

## 🛠 Tech Stack

Frontend:

* React.js
* Chart.js
* Framer Motion
* Axios

Backend:

* FastAPI
* Scikit-learn
* SQLite
* JWT Authentication

## ⚙️ Installation

### Backend

```
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

### Frontend

```
cd frontend
npm install
npm start
```
